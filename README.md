# Business Communication - Values, Vision, and Ventures

A business communication document outlining the values, vision, and ventures of NineOneSix Systems and related entities.

## GitHub Pages Setup

This repository is configured for GitHub Pages. The site will be available at:
- `https://nine-one-six-systems.github.io/[repository-name]/`

### Local Development

Simply open `index.html` in your web browser to view the document locally.

### Deployment

Once pushed to the `main` branch, GitHub Pages will automatically deploy the site. To enable GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Under "Source", select `main` branch
4. Click Save

The site will be live within a few minutes.
